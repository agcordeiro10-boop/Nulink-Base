# Nulink Base — CRM Operacional

Sistema de gestão completo para o Nulink: empresas, profissionais, pipeline, tarefas, métricas e automações.

## Stack
- Next.js 14 (App Router)
- Supabase (PostgreSQL + Auth)
- Vercel (Deploy)
- Recharts (Gráficos)

## Módulos
- Dashboard com métricas em tempo real
- Gestão de Empresas/Leads
- Gestão de Profissionais
- Pipeline Kanban
- Tarefas
- Histórico de Interações
- Métricas & Gráficos
- Relatórios com exportação CSV
- Automações e follow-ups

## Setup
Criar `.env.local`:
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```
